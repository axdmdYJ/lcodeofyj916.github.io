Problem: 344. 反转字符串

思路

定义两个指针,分别初始化为指向左端点和指向右端点,

当l < r的时候,说明二者已将字符转遍历完

复杂度

  • 时间复杂度: O(n)
  • 空间复杂度: O(1)

Code

class Solution {
    public void reverseString(char[] s) {
                int l = 0;
                int r = s.length - 1;

               while(l<r){
                char temp = s[l];
                s[l] = s[r];
                s[r] = temp;
                l++;
                r--;
                }
    }
}

Problem: 541. 反转字符串 II

思路

因为每计数至 2k 个字符，就反转这 2k 字符中的前 k 个字符,所以可以定义一个指针 i ,来从起点,以2k的长度来遍历,

在遍历的同时,需要判断此时的位置是否超出字符串的长度

比较容易疑惑的就是边界问题

例如:if(i + k <= ch.length)在这个判断条件下,ch.length - 1也能通过,

这是因为,这一步边界的判断是为了操作最后剩余的数,事实上,在for循环里,循环条件便是i < ch.length,所以到最后一步后,操作是统一的,都是 reverse(ch, i, ch.length-1); reverse(ch, i, ch.length-1);这里的reverse是自己实现的左闭右闭原则

复杂度

  • 时间复杂度: O(n)
  • 空间复杂度: O(1)

Code

class Solution {
    public String reverseStr(String s, int k) {
        char[] ch = s.toCharArray();

        for(int i = 0; i < ch.length; i += 2*k){
            if(i + k <= ch.length){
                reverse(ch , i, i+k-1);
                continue;
            }
            reverse(ch, i, ch.length-1); //这里自己实现的reverse是左闭右闭原则,所以ch.length-1
        }
        return new String(ch);
    }

    public void reverse(char[] ch, int i, int j){
        for(; i <= j; i++,j--){
            char temp = ch[i];
            ch[i] = ch[j];
            ch[j] = temp;
        }
    }
}

Problem: 题目：剑指Offer 05.替换空格

思路

因为要把空格替换为“%20”, 所以容易知道:

数组的长度是需要扩大的,可以在原数组增大,或者创建一个新的数组

可以定义两个指针,从新数组最后往前遍历,和从老数组后往前遍历,这样可以省去从前往后每次遇到空格需要挪动数组的操作

将 j遍历的字符赋值给i, 如果遇到空格, i就独自往前赋值为“%20”, 然后和j继续重复以上操作

复杂度

  • 时间复杂度：O(n)
  • 空间复杂度：O(1)

Code

C++

class Solution {
public:
    string replaceSpace(string s) {
        int count = 0; // 统计空格的个数
        int sOldSize = s.size();
        for (int i = 0; i < s.size(); i++) {
            if (s[i] == ' ') {
                count++;
            }
        }
        // 扩充字符串s的大小，也就是每个空格替换成"%20"之后的大小
        s.resize(s.size() + count * 2);
        int sNewSize = s.size();
        // 从后先前将空格替换为"%20"
        for (int i = sNewSize - 1, j = sOldSize - 1; j < i; i--, j--) {
            if (s[j] != ' ') {
                s[i] = s[j];
            } else {
                s[i] = '0';
                s[i - 1] = '2';
                s[i - 2] = '%';
                i -= 2;
            }
        }
        return s;
    }
};

Java

可以利用StringBuilder来从前往后遍历,如果遇到空格,就append("%20")

其余就qppend("字符")

class Solution {
    public String replaceSpace(String s) {
            if(s == null){
                return null;
            }

            StringBuilder sb = new StringBuilder();

            for(int i = 0; i < s.length(); i++){
                if(s.charAt(i) == ' '){
                    sb.append("%20");
                }else{
                    sb.append(s.charAt(i));
                }          
            }
               return sb.toString();
    }
}
